# Shreyan-Sequeira-6009CMD---Crop-Yield-Predictor
An Artificial Neural Net for predicting crop yields 

Link to Dataset: https://www.kaggle.com/datasets/samuelotiattakorah/agriculture-crop-yield/data 
Link to video explanation: https://youtu.be/yQWAsCe9xrE 

Instructions for running code:
run the following in command prompt to install the necessary packages:
pip install pandas numpy matplotlib seaborn scikit-learn tensorflow

ensure all installed files are in the same directory before running

Code generated manually is in the file crop_predict.ipynb
Code generated via LLM is in the file AI_Predictor.ipynb
